# 客户端

> 从前端入门的编程，回顾这几年，前端的发展真的是日新月异，各种框架层出不穷，各种技术层出不穷，要选定一些主要的技术栈深入研究，作为生产工作使用，能让工作更得心应手。同时扩展一些其他的技术栈，会让编程更有趣。

## 我的技术栈

- 前端框架：[React](https://react.dev/)、[NextJS](https://nextjs.org/)
- CSS 库： [Tailwindcss](https://tailwindcss.com/)、[Less](https://lesscss.org/)
- UI 组件库：[ShadcnUI](https://ui.shadcn.com/)、[AntDesign](https://ant.design)
- 构建工具：[Webpack](https://webpack.js.org/)、[Rspack](https://rspack.dev/)、[Turbopack](https://nextjs.org/docs/app/api-reference/turbopack)
- 工程管理：[Turborepo](https://turbo.build/repo/docs)、[pnpm](https://pnpm.io/)、
- 代码管理：[Biomejs](https://biomejs.dev/)、[ESlint](https://eslint.org/)、[Prettier](https://prettier.io/)、[Stylelint](https://stylelint.io/)
- 测试：[Playwright](https://playwright.dev/)、[Jest](https://jestjs.io/)
- 3D：[ThreeJS](https://threejs.org/)
